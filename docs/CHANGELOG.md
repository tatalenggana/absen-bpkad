# Changelog

Semua perubahan penting pada proyek ini akan didokumentasikan di file ini.

## [1.0.0] - 2025-11-26

### Fitur Baru

-   Sistem autentikasi (login/register) dengan role-based access
-   Dashboard admin dengan statistik real-time
-   Halaman absensi peserta dengan GPS geofencing
-   Check-in dan check-out dengan capture foto
-   Laporan absensi dengan filter multiple
-   Pengaturan sistem dinamis untuk jam dan lokasi absensi

### Peningkatan

-   UI/UX yang modern dan responsif
-   Dark sidebar theme dengan gradient cards
-   Real-time distance calculation
-   Status detection otomatis (Hadir, Terlambat, Absen)
-   Admin dapat mengubah lokasi geofencing tanpa coding

### Bug Fixes

-   Perbaikan validasi geofencing
-   Fix typo "Sekretaria" menjadi "Sekretariat"
-   Perbaikan pagination di halaman laporan
-   Fix routing untuk admin settings

### Deprecations

-   Emoji dari dokumentasi dan interface dihapus untuk konsistensi
